# JSON

## JSON이란

- JSON(JavaScript Object Notation)은 javascript 객체 문법으로 구조화된 데이터를 표현하기 위한 표준 포맷이다.
- JSON은 javascript 객체 문법을 다르는 문자열이다.
- key : value 형태이다.

<hr>

## JSON 자료형

- 문자열 (String)
- 수 (Number)
- 참/거짓 (Boolean)
- 배열 (Array)
- 객체 (Object)
- null

<hr>

## JSON 장점

- 텍스트로 이루어져 있어 사람과 컴퓨터 모두 읽고 쓰기 쉽다.
- 프로그래밍 언어와 플랫폼에 독립적이므로, 서로 다른 시스템간에 객체를 교환하기에 좋다.
- 가볍다

<hr>

## JSON 예시

```json
{
  "name": "jangwoo",
  "like": {
    "아령": {
      "weight": "10kg",
      "feature": "육각형"
    },
    "바나나": {
      "color": "초록색"
    }
  }
}
```
